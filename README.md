# Portal Académico

## Herramientas utilizadas

- **Frontend:** Angular 21 (v21.2.7)
- **Backend:** Spring Boot (próximamente)
- **Contenerización:** Docker / Docker Compose
- **Control de versiones:** GitHub Desktop

## Puertos

| Servicio  | Puerto                  |
|-----------|-------------------------|
| Frontend  | http://localhost:4200   |
| Backend   | http://localhost:8080 (próximamente) |
